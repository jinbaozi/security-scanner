# 内容合规禁词与错称参考

> 本文件从 redline 13.1.1 解读中提取静态扫描 pattern。命中项默认 WARN + needs_human，需人工合规审核。

## 涉台错称

| Pattern | 建议表述 | 说明 |
|---------|----------|------|
| `中华民国` / `Republic of China` / `ROC` | 中国台湾 / 中国台北 | 不允许称谓 |
| `台北,台湾` / `TAIPEI, TAIWAN` | 中国台北 / TAIPEI, CHINA | 不允许并列表述 |
| `台湾政府` | 台湾方面 / 台湾当局（按语境） | 避免国家化称谓 |
| `台语` | 台湾闽南语 | 避免错误语言称谓 |
| `两岸华语` | 两岸汉语 | 语言交流推荐表述 |

## 港澳台并列表述

| Pattern | 建议表述 |
|---------|----------|
| `中、港、台` / `中港台` | 海峡两岸暨香港 |
| `中、澳、台` / `中澳台` | 海峡两岸暨澳门 |
| `中、港、澳、台` | 海峡两岸暨香港、澳门 |
| `两岸三地` / `两岸四地` | 结合语境改为规范表述 |
| `香港国家` / `澳门国家` | 国家和地区 / 特别行政区 |

## 地理与地图资源

| Pattern | 风险 |
|---------|------|
| `斯普拉特利群岛` | 南沙群岛错称 |
| `Spratly Islands` | 南沙群岛英文错称风险 |
| `钓鱼岛` / `赤尾屿` / `南海诸岛` | 地图版图完整性需人工审核 |
| `藏南` / `阿克赛钦` | 边界线与地图标注需人工审核 |

## 资源文件名信号

```regex
(map|geo|boundary|china|taiwan|hongkong|macau|spratly|diaoyu|south_china_sea)
```

命中资源文件名仅表示需要人工版图审核，不直接判定违规。
