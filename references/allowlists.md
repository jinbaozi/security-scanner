# 白名单与例外规则

> 本文件定义扫描过程中的白名单和例外规则。用户可自行扩展，无需修改 SKILL 核心逻辑。

## 1. 公网地址白名单

### 1.1 标准协议命名空间（自动豁免）

以下域名在 SOAP/XML/WSDL 等标准协议中必须使用，自动豁免：

- `*.w3.org` — W3C 标准命名空间
- `*.xmlsoap.org` — SOAP 协议标准命名空间
- `schemas.microsoft.com` — Microsoft 标准 Schema
- `*.oasis-open.org` — OASIS 标准组织
- `xmlns.xmlsoap.org` — SOAP XML 命名空间
- `schemas.xmlsoap.org` — SOAP Schema
- `www.w3.org` — W3C 标准引用
- `xml.apache.org` — Apache XML 命名空间
- `docs.oasis-open.org` — OASIS 文档引用

### 1.2 语言/框架强制引用（自动豁免）

以下域名因语言或框架限制无法去除，自动豁免：

- `github.com` — Go 语言 import 路径、npm 包引用
- `gitee.com` — Go 语言 import 路径
- `go.uber.org` — Go Uber 库
- `golang.org` — Go 标准扩展库
- `pkg.go.dev` — Go 包文档引用
- `godoc.org` — Go 文档引用
- `registry.npmjs.org` — npm 包注册表
- `pypi.org` — Python 包索引
- `maven.apache.org` — Maven 配置引用
- `repo1.maven.org` — Maven 中央仓库

### 1.3 回环与私有地址（自动豁免）

- `127.0.0.1`, `localhost`, `::1` — 回环地址
- `10.0.0.0/8` (10.x.x.x) — RFC 1918 私有地址 A 类
- `172.16.0.0/12` (172.16-31.x.x) — RFC 1918 私有地址 B 类
- `192.168.0.0/16` (192.168.x.x) — RFC 1918 私有地址 C 类
- `0.0.0.0` — 通配地址
- `169.254.x.x` — 链路本地地址

## 2. 第三方/生成代码排除规则

### 2.1 目录名匹配（自动排除）

- `vendor/` — Go 依赖
- `third_party/`, `thirdparty/` — 第三方代码
- `external/` — 外部依赖
- `generated/`, `gen/` — 自动生成代码
- `node_modules/` — Node.js 依赖
- `__pycache__/` — Python 缓存
- `.git/`, `.svn/`, `.hg/` — 版本控制
- `target/`, `build/`, `dist/` — 构建输出
- `.venv/`, `venv/` — Python 虚拟环境

### 2.2 文件内容匹配（自动排除）

文件头部包含以下标记的，视为自动生成代码：

- `AUTO-GENERATED FILE`
- `DO NOT EDIT`
- `Code generated by`
- `Automatically generated`
